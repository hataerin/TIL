# 사업 아이템 찾기

네이버 데이터랩 > 쇼핑 인사이트

- 분야통계
  - 전년도의 [판매 예상월] 검색
  - 상품의 1년 기록 : 계절을 타지 않는 것
  - 연령별, 성별, 접근도구가 50%



네이버 광고 > 키워드 도구!?

- 키워드 입력
  - 월간 검색수 

네이버 쇼핑 > 키워드 입력 > 등록된 상품 수

- 키워드입력 > 인기쇼핑몰 > 소호몰 > 1등 최강자
  - 1등 쇼핑몰 > 전체상품 > 누적판매순 > 1등상품 > 전주 배송상품수 : N개 >  
    - 1등이 N개파는 시장인것을 알 수 있음



검색수 > 등록된 상품수



네이버 쇼핑 >  인기 쇼핑몰
